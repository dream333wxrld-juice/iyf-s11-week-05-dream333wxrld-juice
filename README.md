## Project Description
Learned how to make web pages interactive using the DOM and JavaScript event handling. Built a fully functional To-Do List application with add, toggle complete, delete, and filter functionality. Also practiced DOM selection, traversal, content manipulation, and various event types.

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Features
- Interactive To-Do List with add, toggle, delete, and filter functionality
- Real-time item count tracking
- Clear completed tasks button
- DOM manipulation exercises (selection, traversal, content editing)
- Event handling exercises (click, keyboard, form events)
- Daily challenges including color changer, dynamic element creator, and image remover

## How to Run
1. Clone this repository
2. Open `index.html` in your browser to view the To-Do List
3. Open `day1.html`, `day2.html`, `day3.html` etc. for daily challenges

## Lessons Learned
Learned how to select and manipulate DOM elements using various methods like querySelector and getElementById. Understood event delegation and how it improves performance when handling dynamic elements. Gained experience building a complete interactive application from scratch.

## Challenges Faced
Understanding event delegation took some practice, especially handling clicks on dynamically created elements. Also learned the importance of using event.target to identify which element triggered an event.
