## Project Description
Learned how to make web pages interactive using the DOM and JavaScript event handling. Built a fully functional To-Do List application with add, toggle complete, delete, and filter functionality. Also completed DOM selection, traversal, content manipulation, and event handling exercises including event bubbling, delegation, and form validation.

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Features
- Interactive To-Do List with add, toggle, delete, and filter functionality
- Real-time item count tracking and clear completed tasks
- DOM manipulation exercises (selection, traversal, content/attribute/style editing)
- Event handling exercises (click, keyboard, form, bubbling, delegation)
- Click counter with increment, decrement, and reset
- Delegated task list using a single event listener
- Real-time form validation
- Daily challenges: color changer, dynamic element creator, image remover, content copier, dark mode toggle

## How to Run
1. Clone this repository
2. Open `index.html` in your browser to view the To-Do List
3. Open `dom-practice.html` for DOM exercises (Lesson 9)
4. Open `events-practice.html` for event handling exercises (Lesson 10)
5. Open `day1.html` through `day5.html` for daily challenges

## Lessons Learned
Learned how to select and manipulate DOM elements using various methods like querySelector and getElementById. Understood event delegation and how it improves performance when handling dynamic elements, especially in the task list. Practiced event bubbling and how events propagate from child to parent elements.

## Challenges Faced
Understanding event delegation took practice, especially handling clicks on dynamically created elements without attaching individual listeners to each one. Also learned the importance of using event.target versus event.currentTarget to correctly identify which element triggered an event.
